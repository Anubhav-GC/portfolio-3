# Certificate Files

Add your certificate PDF or image files to this folder.

## Expected Files

Place the following certificate files in this directory:

1. **Infosys_Python_Part1.pdf** - Infosys Springboard Programming Fundamentals
2. **GeeksforGeeks_Python_Course.pdf** - GeeksforGeeks Python Course Certificate
3. **Accenture_AI_Course.pdf** - Accenture AI Course Certificate
4. **Yuva_AI_For_All.pdf** - Yuva AI for All

## Supported Formats

- PDF files (.pdf)
- Image files (.jpg, .jpeg, .png)

## How to Use

1. Save your certificate files with the exact names mentioned above
2. Place them in this `certificates` folder
3. Users will be able to download them by clicking the "Download" button on your portfolio
